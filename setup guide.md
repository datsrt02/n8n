# N8N + LLM + SD in one shot!!!

**Quick Install Command (Non-IT)**

```bash
Server Ubuntu:
- VPS Ubuntu Có GPU: curl -L https://raw.githubusercontent.com/datsrt02/n8n/refs/heads/main/install.sh | sh
- VPS Ubuntu Không GPU: curl -L https://raw.githubusercontent.com/datsrt02/n8n/refs/heads/main/install_noai.sh | sh

Server Debian
- VPS Debian Không GPU: curl -L https://raw.githubusercontent.com/datsrt02/n8n/refs/heads/main/n8n_install_noai_debian.sh | sh

Bật Ngrok
- Chạy lại n8n có Ngrok: sh <(curl -L https://raw.githubusercontent.com/datsrt02/n8n/refs/heads/main/start_with_ngrok.sh) (chỉ chạy sau khi chạy 1 trong 2 lệnh trên)
- Chạy lại n8n có Ngrok trên thiết bị Pi/TvBox: sh <(curl -L https://raw.githubusercontent.com/datsrt02/n8n/refs/heads/main/start_with_ngrok_pi.sh) (chỉ chạy sau khi chạy 1 trong 2 lệnh trên)
```
